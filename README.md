## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/MeghanaPidaparthi/Meghana-portfolio.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm start
    ```
